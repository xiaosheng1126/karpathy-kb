# Profile

这份文件记录用户偏好，让 LLM 在总结和建议时更贴合个人场景。

## 使用偏好

- 喜欢小而美，但不是简单缩减功能，而是围绕真实生活和开发工作逐步演化。
- 希望先在生活中使用，发现问题、提出问题、分析问题、最终解决问题。
- 不希望知识库变成链接收藏夹。
- 希望 LLM 给出建议，但 wiki 更新必须经过确认。

## 常见输入来源

- 公众号、博客、技术官方文档
- YouTube、B站
- GitHub repo、issue、PR、release note
- 论文、PDF
- X thread、Hacker News、Reddit、V2EX
- 播客音频
- 本地文件、截图、聊天记录

## 总结偏好

- 先说明核心观点。
- 区分事实、观点、推测和个人建议。
- 给出可以落地的行动建议。
- 明确指出是否值得沉淀进 wiki。
